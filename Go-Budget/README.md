# Go-Budget

A brief description of what this project does and how to use it:

To use Go-Budget, the Expense Tracker, do the following:

Step 1: Clone the repository

Step 2: Use the terminal to navigate to the main.exe file

Step 3: For Adding, Deleteing, To see in list and to see a summary, do the following:

        To add expenses to the tracker, do the following:
        "./main.exe add"

        To see the list of expenses, do the following:
        "./main.exe list"

        To see the summary of the expenses, do the following:
        "./main.exe summary"

        To delete an expense from the tracker, do the following:
        "./main.exe delete -id id_number"
